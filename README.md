## 📚 Descrição
Este é o portal de acompanhamento de aulas de inglês para fins específicos (ESP) da **Monica Araujo**. Aqui você encontra:
- **Histórico de Aulas:** Todas as lições anteriores com temas e sumários.
- **Glossário Interativo:** Busca rápida por termos técnicos e do dia a dia vistos em aula.
- **Próxima Aula:** Card dinâmico com o agendamento da próxima sessão.

## 🛠️ Tecnologias
- HTML5 / CSS3 / JavaScript (Vanilla)
- JSON para gestão dinâmica de dados
- GitHub Pages para hospedagem automática
