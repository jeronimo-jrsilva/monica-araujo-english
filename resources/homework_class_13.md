##  Part 2: Modal Verbs Challenge (10 Exercises)
*Complete the sentences with: Must, Should, Can, Would, or Could.*

1. You ________ (Obligation) follow the RDC 50 standards for this project.
2. I ________ (Ability) design complex layouts using BIM software.
3. We ________ (Suggestion) change the lighting in the waiting area.
4. ________ you please send me the blueprints by Friday?
5. If we had more space, we ________ (Hypothesis) add a garden.
6. Architects ________ (Requirement) be registered to sign technical drawings.
7. You ________ (Advice) check the plumbing redundancy before the inspection.
8. I ________ (Hypothesis) hire more assistants if I had the budget.
9. ________ the client approve this messy layout? I don't think so.
10. We ________ (Obligation) ensure the imaging room is fully lead-shielded.

## ❄️ Part 3: The Conditional Mix (0, 1st, and 2nd) (10 Exercises)
*Identify the type (0, 1, or 2) and complete the verb in the correct form.*

1. (**Type 0**) If you heat water to 100 degrees, it ________ (boil).
2. (**Type 1**) If the client ________ (approve) the budget tomorrow, we will start.
3. (**Type 2**) If I ________ (be) you, I would talk to the contractor.
4. (**Type 1**) If it rains tomorrow, they ________ (not/work) on the roof.
5. (**Type 0**) If the software ________ (crash), you lose your progress.
6. (**Type 2**) If we ________ (have) more land, we would build a parking lot.
7. (**Type 1**) If she ________ (not/study) the code, she will fail the inspection.
8. (**Type 2**) If the HVAC system ________ (fail), the surgery would be cancelled.
9. (**Type 0**) If the red light ________ (blink), the system is overloaded.
10. (**Type 1**) If you ________ (finish) the draft today, I will review it tonight.
