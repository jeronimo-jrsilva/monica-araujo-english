## ⚡ 2. Grammar Quick Ref: How to use it?
- **Past Simple:** Use para algo que ACABOU e tem DATA. (*I built it in 2020.*)
- **Present Perfect:** Use para EXPERIÊNCIA ou STATUS. (*I have built many things.*)
