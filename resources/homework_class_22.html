Homework Class 22: Hospital Lighting & Intensifiers
Part 1: Technical Vocabulary

Choose the correct word to complete the technical description:

01. The nurse requested a (fixture / furniture) for the wall to hold the new medical monitor.
02. During the surgery, we noticed a slight (flicker / glare) in the main light; we need to check the wiring.
03. To improve patient comfort, we installed a (dimmer / motion sensor) in every private room so they can adjust the light.
04. The (lighting / lightning) in the corridor is too warm; we should switch to cool LEDs.
05. We use (emergency / circadian) lighting to help patients maintain a natural sleep cycle in windowless rooms.
Part 2: Adverbs of Intensity

Rewrite the sentences using the adverb in parentheses to change the intensity:

01. The new LED system is brighter than the old one. (Significantly)
02. The emergency lights are visible during the day. (Barely)
03. The light in the laboratory is warm. (Slightly)
04. The operating room must be bright. (Extremely)
05. This hallway is dark compared to the entrance. (Relatively)
Part 3: Translation Challenge

Translate the following technical requirements to English, using the vocabulary and intensifiers from Class 22:

01. "Nós precisamos reduzir significativamente o ofuscamento (glare) na recepção."
02. "A luminária (fixture) está piscando (flickering) levemente (slightly)."
03. "O sensor de movimento (motion sensor) é extremamente sensível."
