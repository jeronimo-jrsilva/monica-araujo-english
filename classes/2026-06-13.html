Class 38: The Endurance Edition - 40-Drill Challenge

📅 Lesson Info

  • Date: 2026-06-13
  • Topic: The Endurance Edition: 40-Drill Challenge

Exaustão Produtiva. Esta aula é um teste de resistência focado na automação de 4 estruturas gramaticais essenciais em cenários de alta pressão técnica. A aula continua na terça-feira.

1. The Causative Machine (10 Scenarios)

Instant delegation of fixes and maintenance.

  1. "The server room is overheating." → "I'll have the AC fixed."
  2. "The hospital signs are in Portuguese only." → "I'll have them translated."
  3. "The MRI machine is vibrating." → "I'll have it inspected."
  4. "The walls are dirty." → "I'll have them repainted."
  5. "The software is crashing." → "I'll have it updated."
  6. "The parking lot lights are off." → "I'll have the bulbs replaced."
  7. "The security protocol is old." → "I'll have it revised."
  8. "The floor is slippery." → "I'll have it cleaned."
  9. "The windows are broken." → "I'll have them repaired."
  10. "The inventory is low." → "I'll have more supplies ordered."

2. The Time Machine (10 Scenarios - CONTINUAÇÃO TERÇA)

Mastering the contrast between Past Simple and Present Perfect.

  1. "The inspector arrived at the site ten minutes ago."
  2. "We have already implemented the new safety measures."
  3. "I wrote the punch list yesterday afternoon."
  4. "Has she already sent the proposal?"
  5. "They looked into the leak two hours ago."
  6. "Have you ever overseen a project this big?"
  7. "The board cancelled the meeting yesterday."
  8. "I have just talked to the inspector."
  9. "We had a major power outage in 2024."
  10. "Have you ironed out the details yet?"

3. The Compliance Machine (10 Scenarios)

Reacting to violations using MUST / SHOULD / CAN'T.

  1. "Smoking near oxygen tanks?" → MUST STOP.
  2. "Lead time too long?" → SHOULD FIND NEW SUPPLIER.
  3. "Forgotten ID badge?" → MUST GET IT.
  4. "Budget almost empty?" → SHOULD CUT COSTS.
  5. "Photos in ICU?" → CAN'T DO THAT.
  6. "Locked fire exit?" → MUST UNLOCK.
  7. "Confusing blueprints?" → SHOULD CLARIFY.
  8. "No safety goggles?" → MUST WEAR.
  9. "Deadline tomorrow?" → SHOULD CHECK TODAY.
  10. "Unauthorized access?" → MUST LEAVE.

4. The Logical Machine (10 Scenarios)

Instant What-If scenarios for project management.

  1. "If the audit starts late, it will delay our weekend."
  2. "If I had more time, I would oversee the whole process."
  3. "If the materials arrive today, we will start the assembly."
  4. "If she were the manager, she would hire more subcontractors."
  5. "If the client rejects the plan, we will revise it."
  6. "If the elevator failed again, I would demand a refund."
  7. "If we find a bottleneck, we will report it."
  8. "If it were easy, everyone would do it."
  9. "If the inspector comes now, he will find issues."
  10. "If you spoke to the CEO, what would you say?"

5. Vocabulary Mastery (New)

  • Foundation: Fundação / Base.
  • Inventory: Inventário / Estoque.
  • MRI: Ressonância Magnética.
  • Overheating: Superaquecimento.
  • Parking lot: Estacionamento.
  • Security protocol: Protocolo de segurança.
  • Slippery: Escorregadio.
  • Vibrating: Vibrando.
