Class 34: The Architect's Foundation: Basic Structures & Initial Connectors

📅 Lesson Info

  • Date: 2026-06-04
  • Topic: The Architect's Foundation: Basic Structures & Initial Connectors

Foco em precisão estrutural (auxiliares) e transição para conectores lógicos de projeto.

🎯 1. Warm-up: The Daily Stand-up

Prompt: “Tell me about your day at the office. What projects did you touch today?” (Focus: Do/Does/Did + 3rd Person “S” + Past Tense)

🏗️ 2. Sound Lab: Structural Rhythm (Auxiliaries)

Objective: Reflexive use of negatives and questions.

Drills: - “I need the blueprint.” -> “I DON’T need the blueprint.” - “She wants a meeting.” -> “DOES she want a meeting?” - “The material arrived.” -> “DID the material arrive?” - “We have time.” -> “We DON’T have time.”

🧱 3. Core Drill: The Logic Bricks

Objective: Building sentences with logical connectivity.

A. Simple Bricks:

  1. The client / Like / Modern Style -> “The client likes the modern style.”
  2. We / Not / Finish / The floor plan / Yesterday -> “We didn’t finish the floor plan yesterday.”
  3. The Hospital / Need / More space / For the ICU -> “The hospital needs more space for the ICU.”

B. Connecting Bricks (Sentence Linkers):

  1. HEPA Filter / Expensive / But / Necessary -> “The HEPA filter is expensive BUT it is necessary.”
  2. The budget / Cut / So / We / Change / The material -> “The budget was cut, SO we changed the material.”
  3. The site / Be / Small / That’s why / We / Use / Glass -> “The site is small, THAT’S WHY we use glass.”

🔬 4. Contextual Injection: The Project Pitch

Objective: Precision through Relative Clauses (Who, Which, That).

  • “This is the clinic THAT handles pediatric care.”
  • “He is the engineer WHO manages the HVAC system.”
  • “That is the material WHICH provides better hygiene.”

🏆 5. Final Challenge: The Non-Stop Minute

Topic: “Hospital Safety” Rules: 1 minute talk using: “Because”, “So”, “Which”.
