## ⚡ 1. The Causative Machine: Delegation Power
*Grammar: HAVE + SOMETHING + DONE (V3)*

| Scenario | Response (Reflex) |
| :--- | :--- |
| "The server room is overheating." | "I'll **have** the AC **fixed**." |
| "The hospital signs are in Portuguese only." | "I'll **have** them **translated**." |
| "The MRI machine is vibrating." | "I'll **have** it **inspected**." |
| "The walls are dirty." | "I'll **have** them **repainted**." |
| "The software is crashing." | "I'll **have** it **updated**." |
| "The parking lot lights are off." | "I'll **have** the bulbs **replaced**." |
| "The security protocol is old." | "I'll **have** it **revised**." |
| "The floor is slippery." | "I'll **have** it **cleaned**." |
| "The windows are broken." | "I'll **have** them **repaired**." |
| "The inventory is low." | "I'll **have** more supplies **ordered**." |
