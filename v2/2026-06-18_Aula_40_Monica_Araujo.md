## 🛡️ 2. The Rule Book: Technical Compliance (Modals Drill)
*Objetivo: Alternar entre obrigatoriedade (must), conselho (should) e possibilidade (could/might).*

1.  **You must install lead-lining in the X-Ray room.**
    *   (should / extra outlets / MRI suite)
    *   (cannot / standard glass / laboratory)
2.  **The technician should check the filters every month.**
    *   (must / verify / shielding / year)
    *   (might / inspect / HVAC / week)
3.  **We could use recycled wood for the main lobby.**
    *   (should not / use / cheap vinyl / patient rooms)
    *   (might / consider / natural stone / entrance)
4.  **The inspector might reject the ventilation plan.**
    *   (will / approve / safety exits)
    *   (must / review / fire protocols)
5.  **Staff must wear safety vests on the construction site.**
    *   (should / carry / ID badges / hospital)
    *   (can / use / helmets / site)
6.  **The client cannot change the structural design now.**
    *   (must / approve / material list / today)
    *   (should / sign / contract / immediately)
7.  **We should iron out the snags before the audit.**
    *   (must / finish / punch list / Monday)
    *   (could / fix / lighting / tonight)
8.  **The board might increase the budget for the facade.**
    *   (should / reduce / costs / landscaping)
    *   (will / maintain / original / plan)
9.  **I must brief the team about the new regulations.**
    *   (should / warn / subcontractor / risks)
    *   (could / email / board / updates)
10. **You should not place the oxygen tanks near the exit.**
    *   (must / keep / fire extinguishers / visible)
    *   (can / store / cleaning products / here)
