1. THE CAUSATIVE MACHINE: Delegation Power
Structure: HAVE + SOMETHING + DONE (V3)
"Use this when someone else does the job for you."

Drills:
- Server room overheating? -> "I'll have the AC fixed."
- Hospital signs in PT? -> "I'll have them translated."
- MRI vibrating? -> "I'll have it inspected."
- Walls dirty? -> "I'll have them repainted."
- Software crashing? -> "I'll have it updated."
- Parking lights off? -> "I'll have the bulbs replaced."
- Security protocol old? -> "I'll have it revised."
- Floor slippery? -> "I'll have it cleaned."
- Windows broken? -> "I'll have them repaired."
- Inventory low? -> "I'll have more supplies ordered."

2. TIME MASTERY: Past Simple vs. Present Perfect
Reflex Practice:

1. "Did you call the subcontractor this morning?"
   (Specific time: This morning)
2. "I have never worked with this material before."
   (Experience: Never)
3. "We finished the foundation last month."
   (Specific time: Last month)
